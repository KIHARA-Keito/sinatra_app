# sinatra_app

提出用のメモアプリです。
cloneして対象フォルダに移動後、以下を実行すればローカルにて動作確認いただけます。

## gem のインストール

```
bundle install
```

## アプリケーション起動

```
bundle exec ruby app.rb
```

下記にアクセスしてください。

http://localhost:4567/
